# sixlmx

sixlmx — образовательная организация и пространство для учебных проектов.

Здесь будут собираться репозитории, связанные с обучением, разработкой, практикой и развитием образовательных продуктов.

## Основные направления

- учебные проекты;
- разработка образовательных инструментов;
- практика командной работы;
- развитие LMS- и EdTech-решений;
- документация и инженерные регламенты.

## Принципы работы

- аккуратный Git-flow;
- понятная документация;
- безопасная работа с доступами;
- развитие через маленькие проверяемые изменения;
- обучение через практику.
